## 🎯 Business Problem
Perusahaan retail ingin meningkatkan response rate marketing campaign (saat ini 14.9%).

## 💡 Key Insights
- Wine: $680K+ spending (dominan)
- High Income customers: response rate lebih tinggi
- Catalog channel: highest value per transaction
- Outlier: income &gt;$600K tapi spending rendah (churn risk)

## 🛠️ Tech Stack
- Python, Pandas, NumPy
- Matplotlib, Seaborn, Plotly
- Streamlit

## 📁 Dataset
Customer Personality Analysis (Kaggle)
- 2,240 customers
- 22 attributes
